## ✨ Key Features

- **🔍 Verb Explorer**: Instantly conjugate any English verb into its 5 primary lexical forms ($V_1$ to $V_5$).
- **📊 Tense Matrix**: Interactive grid showcasing past, present, and future timelines across four aspects (Simple, Continuous, Perfect, Perfect Continuous).
- **💡 Conjugation Focus Modal**: Click any cell in mobile viewports to open a fully blurred backdrop drawer with Structural Equations, usage purposes, and Person Agreement Checks.
- **🧠 Grammar AI**: Input sentences and receive detailed correctness diagnostics, correction logs, and sentential tense break-downs powered by Gemini.

## 🚀 Quick Start

### 1. Prerequisites
Make sure you have [Node.js](https://nodejs.org) installed on your system.

### 2. Installation
Clone the repository and install dependencies:
```bash
npm install
```

### 3. Environment Setup
Configure your API Key. Create a `.env` file in the root directory:
```env
GEMINI_API_KEY=your_gemini_api_key_here
```

### 4. Running the App
Launch the Vite & Express development server:
```bash
npm run dev
```
Open **[http://localhost:3000](http://localhost:3000)** in your browser!
